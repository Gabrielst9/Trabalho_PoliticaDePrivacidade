Termo de Uso dos Sistemas Internos

Código: TUSI-012

Emissão: 01/10/2024

Versão: 1.00

Classificação: Uso interno

Aprovado por: Gabriel Tavares

1. Objetivo

Estabelecer regras e diretrizes para o uso dos sistemas internos da SecurityCredit, garantindo segurança, integridade e conformidade.

2. Aceitação do Termo

Ao acessar e utilizar os sistemas internos, o colaborador concorda em cumprir todas as disposições deste Termo de Uso.

3. Escopo

Este termo aplica-se a todos os colaboradores que utilizam os sistemas internos da SecurityCredit, incluindo, mas não se limitando a, softwares de gestão, bancos de dados e plataformas de comunicação.

4. Responsabilidades do Usuário

Os colaboradores devem:

  • Uso Adequado: Utilizar os sistemas apenas para fins profissionais.
  • Confidencialidade: Manter a confidencialidade das informações acessadas.
  • Relatar Anomalias: Notificar imediatamente a equipe de TI sobre quaisquer problemas ou incidentes.

5. Proibições

É estritamente proibido:

  • Acesso Não Autorizado: Tentar acessar áreas do sistema que não são permitidas.
  • Alteração de Dados: Modificar ou excluir dados sem autorização adequada.
  • Uso Malicioso: Utilizar os sistemas para realizar atividades ilegais ou prejudiciais.

6. Segurança da Informação

Os colaboradores devem:

  • Senhas: Criar senhas fortes e mantê-las em segredo.
  • Autenticação: Utilizar mecanismos de autenticação fornecidos pela SecurityCredit para acesso seguro.

7. Monitoramento

A SecurityCredit se reserva o direito de monitorar o uso dos sistemas para garantir a conformidade com este termo e proteger os ativos da informação.

8. Consequências de Violações

Qualquer violação a este Termo de Uso pode resultar em sanções disciplinares, que podem incluir a rescisão do contrato de trabalho.

9. Revisão e Atualização

Este Termo de Uso pode ser revisado e atualizado periodicamente, e a versão mais recente será disponibilizada aos colaboradores.

10. Contato

Para esclarecimentos sobre este Termo de Uso, entre em contato:

  • E-mail: contato@securitycredit.com.
